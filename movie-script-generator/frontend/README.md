# Movie Script Generator Frontend

This project is a React-based frontend for the Movie Script Generator application, which utilizes Google Generative AI to create movie scripts based on user input.

## Getting Started

To get started with the frontend application, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd movie-script-generator/frontend
   ```

2. **Install dependencies:**
   Make sure you have Node.js installed. Then run:
   ```
   npm install
   ```

3. **Run the application:**
   Start the development server with:
   ```
   npm start
   ```
   This will launch the application in your default web browser at `http://localhost:3000`.

## Folder Structure

- **public/**: Contains the static files, including the main HTML file.
- **src/**: Contains the React components and styles.
  - **components/**: Contains reusable components, including the `ScriptGenerator` component.
  - **styles/**: Contains CSS files for styling the application.
  
## Usage

Once the application is running, you can use the `ScriptGenerator` component to input parameters for generating movie scripts. The generated scripts will be displayed on the screen.
